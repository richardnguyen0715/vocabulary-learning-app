VOCABULARY LEARNING APP 
======================= 
 
CACH SU DUNG: 
1. Chay file start_app.bat 
2. Hoac chay truc tiep VocabularyLearningApp.exe 
   (can dam bao MongoDB da khoi dong) 
 
YEU CAU HE THONG: 
- Windows 10/11 
- MongoDB Community Server 
 
CAI DAT MONGODB: 
1. Tai tu: https://www.mongodb.com/try/download/community 
2. Chon Windows x64, MSI 
3. Cai dat voi cac tuy chon mac dinh 
4. Dam bao "Install MongoDB as a Service" duoc chon
